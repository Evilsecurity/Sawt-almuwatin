📋 ما هو المشروع؟

صوت المواطن منصة مفتوحة لاستطلاع آراء المواطنين حول القضايا المجتمعية. نؤمن بأن كل صوت يساهم في تشكيل أولويات التنمية.

🛡️ الأمان والخصوصية

  • التصويت مجهول المصدر — لا نجمع أي بيانات شخصية
  • يُستخدم Browser Fingerprint لمنع التصويت المزدوج بدلاً من localStorage وحده
  • تحقق مزدوج: محلي (localStorage) + رقمي (fingerprint hash)
  • البيانات المعروضة إحصائية فقط

⚙️ التقنيات المستخدمة

  • Firebase Realtime Database — قاعدة بيانات مشتركة بين جميع المستخدمين
  • Browser Fingerprinting — تحسين أمان التصويت
  • Canvas API — رسوم بيانية دائرية بدون مكتبات خارجية
  • تصدير CSV — لتحليل البيانات خارجياً

🎯 الأهداف

  • قياس اهتمامات المواطنين بالقضايا المختلفة
  • توفير منصة شفافة للتعبير عن الرأي
  • المساعدة في فهم الأولويات المجتمعية
  • تشجيع المشاركة المدنية الفعالة
